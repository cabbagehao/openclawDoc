# `openclaw clawbot`

하위 호환성을 위해 유지되는 레거시 별칭 네임스페이스입니다.

현재 지원되는 별칭:

- `openclaw clawbot qr` (동작은 [`openclaw qr`](/cli/qr) 과 동일)

## 마이그레이션

가능하면 최신 top-level 명령을 직접 사용하세요.

- `openclaw clawbot qr` -> `openclaw qr`
