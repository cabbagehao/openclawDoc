# `openclaw setup`

`~/.openclaw/openclaw.json`과 에이전트 워크스페이스를 초기화합니다.

관련 문서:

- 시작 가이드: [Getting started](/start/getting-started)
- 마법사: [Onboarding](/start/onboarding)

## 예시

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

`setup`을 통해 마법사를 실행하려면:

```bash
openclaw setup --wizard
```
