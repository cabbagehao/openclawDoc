# `openclaw docs`

실시간 문서 인덱스를 대상으로 키워드 검색을 수행함.

## 사용 예시

```bash
# 브라우저 확장 프로그램 관련 문서 검색
openclaw docs browser extension

# 샌드박스 설정(allowHostControl) 관련 문서 검색
openclaw docs sandbox allowHostControl
```
