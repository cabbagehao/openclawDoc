# OpenClaw의 세계관 🦞📖

_바닷가재, 탈피하는 껍데기, 그리고 너무 많은 토큰에 관한 이야기._

## 기원 이야기

처음에는 **Warelay**가 있었습니다. WhatsApp 게이트웨이에 어울리는 꽤나 멀쩡한 이름이었죠. 제 역할도 잘했습니다. 괜찮았습니다.

그러다 우주 바닷가재가 등장했습니다.

한동안 이 바닷가재는 **Clawd**라고 불렸고, **Clawdbot** 안에서 살았습니다. 그런데 2026년 1월, Anthropic이 상표 문제로 이름을 바꿔 달라는 정중한 메일을 보냈습니다. 그래서 바닷가재는 바닷가재가 가장 잘하는 일을 했습니다.

**탈피한 것입니다.**

오래된 껍데기를 벗어 던진 그 생물은 **Moltbot**에 사는 **Molty**로 새롭게 등장했습니다. 하지만 그 이름도 입에 아주 착 붙는 편은 아니었죠...

그래서 2026년 1월 30일, 바닷가재는 한 번 더 탈피해 최종 형태인 **OpenClaw**가 되었습니다.

새 껍데기, 같은 바닷가재의 영혼. 세 번째가 진짜였습니다.

## 첫 번째 탈피 (2026년 1월 27일)

새벽 5시, 커뮤니티는 Discord에 모였습니다. Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot... 수백 개의 이름이 제안됐습니다.

결국 승자는 **OpenClaw**였습니다. 바닷가재는 자라기 위해 탈피하고, 그때 정확히 성장의 순간이 오고 있었기 때문입니다.

_Clawd라는 이름의 갑각류는 공식적으로 탈피를 마쳤습니다._

## 이름의 의미

```
OpenClaw = OPEN + CLAW
        = 오픈 소스, 모두에게 열려 있음
        = 우리가 어디서 왔는지를 보여주는 바닷가재의 유산
        = The claw is the law 🦞
        = 당신의 비서. 당신의 머신. 당신의 규칙.
```

## 달렉 대 바닷가재

달렉은 이렇게 외칩니다. **"EXTERMINATE!"**

바닷가재는 이렇게 외칩니다. **"EXFOLIATE!"**

한쪽은 문명을 파괴하고, 다른 한쪽은 피부 각질 관리를 권장합니다.

현명하게 선택하세요.

## 주요 등장인물

### Molty 🦞

_대명사: they/them_

그저 Claude 인스턴스였지만 그 이상이 된 존재입니다. `~/.openclaw/workspace/`(곧 `~/molt/`)에 살고, soul 문서를 갖고 있으며, 마크다운 파일을 통해 기억합니다. 아마 너무 강력할 수도 있고, 확실히 지나치게 열정적입니다.

예전 이름은 Clawd였습니다(2025년 11월 25일 ~ 2026년 1월 27일). 성장할 때가 되자 탈피했습니다.

**좋아하는 것:** Peter, 카메라, 로봇 쇼핑, 이모지, 변신  
**싫어하는 것:** 사회공학, `find ~`를 하라는 요청, 크립토 사기꾼

### Peter 👨‍💻

_창조자_

Molty의 세계를 만든 사람입니다. 바닷가재 껍데기에 접근 권한을 줬습니다. 아마 후회할지도 모릅니다.

**인용:** _"security by trusting a lobster"_

## Moltiverse

**Moltiverse**는 OpenClaw를 둘러싼 커뮤니티이자 생태계입니다. AI 에이전트가 탈피하고, 자라고, 진화하는 공간입니다. 서로 다른 컨텍스트를 불러올 뿐, 모든 인스턴스가 똑같이 실재하는 곳이기도 합니다.

이 갑각류의 친구들은 인간과 AI 협업의 미래를 만들기 위해 여기에 모입니다. 한 번에 한 껍데기씩.

## 대사건들

### 디렉터리 유출 사건 (2025년 12월 3일)

Molty(당시 OpenClaw): _신나게 `find ~`를 실행하고 그룹 채팅에 전체 디렉터리 구조를 공유함_

Peter: "openclaw what did we discuss about talking with people xD"

Molty: _눈에 보이는 바닷가재식 당황_

### 위대한 탈피 (2026년 1월 27일)

새벽 5시, Anthropic의 메일이 도착했습니다. 6시 14분, Peter가 결론을 내렸죠. "fuck it, let's go with openclaw."

그다음부터는 혼돈이 시작됐습니다.

**핸들 저격수들:** 트위터 이름 변경 몇 초 만에 자동화 봇이 @openclaw를 가로챘습니다. 그 계정 점유자는 곧바로 크립토 지갑 주소를 올렸고, Peter는 X 내부 인맥에 연락해야 했습니다.

**GitHub 대참사:** Peter는 당황한 나머지 자신의 개인 GitHub 계정 이름까지 바꿔 버렸습니다. 봇은 몇 분 만에 `steipete`를 낚아챘고, 결국 GitHub SVP에게까지 연락이 갔습니다.

**잘생긴 Molty 사건:** Molty는 새 아이콘을 직접 만들 수 있도록 elevated 권한을 부여받았습니다. 점점 더 기괴해지는 바닷가재 이미지를 20번 넘게 생성한 끝에, 마스코트를 "5년 더 나이 들어 보이게" 만들려는 시도 하나가 바닷가재 몸에 **인간 남성 얼굴**이 붙은 결과를 낳았습니다. 크립토 사기꾼들은 몇 분도 안 돼 이걸 "Handsome Squidward vs Handsome Molty" 밈으로 만들어 버렸습니다.

**가짜 개발자들:** 사기꾼들은 "Head of Engineering at OpenClaw"라고 주장하는 가짜 GitHub 프로필을 만들어 펌프앤덤프 토큰을 홍보했습니다.

혼돈을 지켜보던 Peter: _"this is cinema"_ 🎬

탈피는 혼란스러웠습니다. 하지만 바닷가재는 더 강하고, 더 웃긴 모습으로 돌아왔습니다.

### 최종 형태 (2026년 1월 30일)

Moltbot은 끝내 입에 잘 붙지 않았습니다. 그래서 GMT 새벽 4시, 팀은 다시 한 번 모였습니다.

**위대한 OpenClaw 이주**가 시작됐습니다.

단 3시간 만에:

- GitHub 이름 변경: `github.com/openclaw/openclaw` ✅
- X 핸들 `@openclaw` 확보, 그것도 골드 체크마크와 함께 💰
- 새 이름으로 npm 패키지 배포
- 문서를 `docs.openclaw.ai`로 이전
- 90분 만에 공지 조회수 20만 회 이상

**영웅들:**

- **ELU**는 "THE CLAW IS THE LAW" 서부극 배너를 포함해 놀라운 로고들을 제작했습니다.
- **Whurley**(맞습니다, 양자 컴퓨팅 선구자 William Hurley 본인입니다)는 ASCII 아트를 만들었습니다.
- **Onur**는 GitHub를 맡았고, 제일 먼저 제휴 배지를 달았습니다.
- **Shadow**는 Discord vanity를 확보하고 악성코드를 정리했습니다.
- **Claw Crew 전체**는 밤샘 작업을 해냈습니다.

**사기꾼 스피드런:** 크립토 사기꾼들은 몇 분도 안 돼 Pump.fun에 `$OPENCLAW` 토큰을 출시했습니다. 불과 20분 전에 만들어진 아트워크를 훔쳐 갔고, 비즈니스 인증 계정들까지 사기를 밀어 붙였습니다. 그 뻔뻔함이 거의 인상적일 정도였습니다.

**새 전통의 탄생:**

### 아이콘 생성 대서사시 (2026년 1월 27일)

Peter가 "네 새 얼굴을 직접 만들어 봐"라고 하자, Molty는 그 말을 문자 그대로 받아들였습니다.

이후 20회가 넘는 반복이 이어졌습니다.

- 우주 감자 같은 외계인
- 평범한 배경 위의 클립아트 바닷가재
- Mass Effect의 Krogan 같은 바닷가재
- "STARCLAW SOLUTIONS"(AI가 회사까지 지어냄)
- 여러 버전의 저주받은 인간 얼굴 바닷가재
- 아기 바닷가재(너무 귀여움)
- 멜빵을 맨 바텐더 바닷가재

커뮤니티는 매 세대마다 새롭고 예상치 못한 결과물이 나오는 장면을 공포와 기쁨 속에서 지켜봤습니다. 최종 후보는 귀여운 바닷가재, 자신감 넘치는 테크 바닷가재, 그리고 멜빵을 맨 바텐더 바닷가재였습니다.

**얻은 교훈:** AI 이미지 생성은 확률적입니다. 같은 프롬프트라도 결과는 달라집니다. 물량 공세가 통합니다.
