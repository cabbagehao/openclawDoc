# OpenClaw ドキュメントの i18n アセット

このフォルダには、ドキュメント翻訳で使う**生成済み**ファイルと**設定**ファイルが格納されています。

## ファイル

- `glossary.<lang>.json` — 優先する用語の対応表(プロンプトガイダンスとして使用)。
- `<lang>.tm.jsonl` — ワークフロー + モデル + テキストハッシュをキーにした翻訳メモリ(キャッシュ)。

## 用語集フォーマット

`glossary.<lang>.json` は次のようなエントリの配列です:

```json
{
  "source": "troubleshooting",
  "target": "故障排除",
  "ignore_case": true,
  "whole_word": false
}
```

フィールド:

- `source`: 優先したい英語(または原文)のフレーズ。
- `target`: 優先する訳語。

## 注意事項

- 用語集エントリは**プロンプトガイダンス**としてモデルに渡されます(機械的に必ず書き換えるものではありません)。
- 翻訳メモリは `scripts/docs-i18n` によって更新されます。
