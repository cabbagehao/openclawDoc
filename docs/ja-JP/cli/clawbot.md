# `openclaw clawbot`

下位互換性のために保持されている従来のエイリアス名前空間。

現在サポートされているエイリアス:

- `openclaw clawbot qr` ([`openclaw qr`](/cli/qr) と同じ動作)

## 移行

最新のトップレベル コマンドを直接優先します。

- `openclaw clawbot qr` -> `openclaw qr`
