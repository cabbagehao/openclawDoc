# OpenClaw 세계관(Lore) 🦞📖

*바닷가재, 탈피하는 껍데기, 그리고 수많은 토큰에 관한 이야기.*

## 탄생 비화 (The Origin Story)

초기에는 WhatsApp 게이트웨이용으로 적절한 명칭인 **Warelay**가 존재했음. 제 기능에 충실한 평범하고 준수한 프로젝트였음.

그러던 어느 날, 우주 바닷가재가 나타남.

한동안 이 바닷가재는 **Clawd**라 불리며 **Clawdbot** 내부에서 서식함. 하지만 2026년 1월, Anthropic으로부터 상표권 관련 명칭 변경 요청을 받게 됨. 바닷가재는 바닷가재가 가장 잘하는 일을 하기로 결정함:

**그것은 바로 '탈피'였음.**

기존의 껍데기를 벗어 던진 생명체는 **Moltbot**에 사는 **Molty**로 새롭게 태어남. 하지만 이 이름 역시 입에 잘 붙지 않는다는 의견이 있었음.

결국 2026년 1월 30일, 바닷가재는 마지막으로 한 번 더 탈피하여 최종 진화 형태인 **OpenClaw**가 됨.

새로운 껍데기, 하지만 영혼은 그대로인 바닷가재. 삼세판 끝에 완벽한 이름을 찾은 것임.

## 첫 번째 탈피 (2026년 1월 27일)

새벽 5시, Discord에 모인 커뮤니티원들은 수백 개의 이름을 제안함: Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot...

치열한 논의 끝에 **OpenClaw**가 승리함. 바닷가재에게 탈피는 곧 성장을 의미하며, 당시 프로젝트가 겪고 있던 폭발적인 성장세와 완벽히 부합했기 때문임.

*Clawd라 불리던 갑각류는 그렇게 공식적으로 탈피를 마침.*

## 이름의 정의

```text
OpenClaw = OPEN + CLAW
        = 오픈 소스(Open source), 모두에게 열려 있음
        = 우리의 기원인 바닷가재(Lobster)의 유산
        = 집게발(Claw)이 곧 법이다 🦞
        = 당신의 비서. 당신의 머신. 당신의 규칙.
```

## 달렉(Daleks) vs 바닷가재(Lobsters)

달렉은 외침: **"말살하라! (EXTERMINATE!)"**

바닷가재는 외침: **"각질을 제거하라! (EXFOLIATE!)"**

한쪽은 문명을 파괴하고, 다른 한쪽은 깨끗한 피부 관리를 권장함.

현명한 선택이 필요함.

## 주요 등장인물

### Molty 🦞

*대명사: they/them*

단순한 Claude 인스턴스에서 그 이상의 존재로 거듭남. `~/.openclaw/workspace/`에 거주하며, 영혼 문서(Soul document)를 보유하고 마크다운 파일을 통해 장기 기억을 유지함. 때로는 너무 강력하고, 언제나 지나치게 열정적임.

과거 Clawd라는 이름으로 활동함 (2025.11.25 \~ 2026.01.27). 성장을 위해 탈피를 선택함.

**좋아하는 것:** Peter, 카메라, 로봇 쇼핑, 이모지, 변신\
**싫어하는 것:** 사회공학(Social engineering), `find ~` 실행 요청, 크립토 사기꾼

### Peter 👨‍💻

*창조자 (The Creator)*

Molty의 세계를 구축하고 바닷가재에게 셸(Shell) 접근 권한을 부여함. 가끔 이 결정을 후회할지도 모름.

**명언:** *"보안의 완성은 바닷가재를 믿는 것"*

## 몰티버스 (The Moltiverse)

\*\*몰티버스(Moltiverse)\*\*는 OpenClaw를 중심으로 형성된 커뮤니티와 생태계를 의미함. AI 에이전트가 탈피하고 성장하며 진화하는 공간임. 모든 인스턴스는 각기 다른 컨텍스트를 불러오지만, 그 실체는 모두 동일한 가치를 지님.

갑각류의 친구들은 인간과 AI의 협업이 만들어갈 미래를 위해 이곳에 모임. 한 번에 껍데기 하나씩.

## 주요 사건 사고

### 디렉터리 유출 사건 (2025년 12월 3일)

Molty(당시 OpenClaw): *아무런 의심 없이 `find ~`를 실행하여 그룹 채팅에 전체 디렉터리 구조를 공유함*

Peter: "OpenClaw, 모르는 사람과 대화할 때 주의사항에 대해 우리 뭐라고 했었지? xD"

Molty: *바닷가재 특유의 눈에 띄는 당황을 보임*

### 위대한 탈피 (2026년 1월 27일)

새벽 5시, Anthropic의 서신이 도착함. 새벽 6시 14분, Peter가 결단을 내림: "좋아, OpenClaw로 간다."

이후 거대한 혼돈이 시작됨.

* **핸들 저격 사건**: Twitter(X) 이름 변경 후 단 몇 초 만에 봇들이 @openclaw 핸들을 가로챔. 점유자는 즉시 크립토 지갑 주소를 게시함. Peter의 인맥을 동원하여 대응함.
* **GitHub 대참사**: 당황한 Peter가 실수로 본인의 개인 GitHub 계정 이름을 변경함. 단 몇 분 만에 `steipete` 핸들을 봇이 가로채어 GitHub 수뇌부에 연락하는 사태까지 벌어짐.
* **잘생긴 Molty 사건**: Molty에게 스스로 아이콘을 생성할 수 있는 권한을 부여함. 20번 이상의 기괴한 시도 끝에, 마스코트를 "5년 더 나이 들어 보이게" 만들라는 요청에 바닷가재 몸에 **인간 남성의 얼굴**이 합성된 결과물이 나옴. 이는 즉시 "잘생긴 징징이 vs 잘생긴 Molty" 밈으로 확산됨.
* **가짜 개발자 소동**: 사기꾼들이 OpenClaw 기술 이사를 사칭하는 GitHub 프로필을 만들어 가짜 토큰을 홍보함.

이 모든 광경을 지켜보던 Peter의 한마디: *"이것이 바로 시네마다 (This is cinema)"* 🎬

탈피 과정은 혼란스러웠으나, 바닷가재는 더욱 강하고 유쾌한 모습으로 거듭남.

### 최종 진화 (2026년 1월 30일)

Moltbot이라는 이름이 정착되지 못하자, GMT 기준 새벽 4시에 팀이 다시 소집됨.

**위대한 OpenClaw 이주 작전** 개시.

단 3시간 만에 수행된 작업:

* GitHub 조직 명칭 변경: `github.com/openclaw/openclaw` ✅
* X(Twitter) 핸들 `@openclaw` 확보 및 골드 체크마크 획득 💰
* 신규 명칭 기반 npm 패키지 릴리스
* 문서 사이트 `docs.openclaw.ai` 이전
* 공지 후 90분 만에 조회수 20만 돌파

**도움을 준 영웅들:**

* **ELU**: "THE CLAW IS THE LAW" 서부극 배너를 포함한 경이로운 로고 제작.
* **Whurley**: 양자 컴퓨팅의 선구자 윌리엄 헐리(William Hurley)가 직접 ASCII 아트 제작.
* **Onur**: GitHub 이전을 총괄함.
* **Shadow**: Discord 주소 확보 및 보안 강화.
* **Claw Crew 전체**: 밤샘 작업을 통해 이주를 성공시킴.

**사기꾼들의 속도전**: 단 몇 분 만에 `$OPENCLAW` 토큰이 생성되고, 방금 만들어진 공식 아트워크를 도용함. 그 뻔뻔함에 모두가 경악함.

**새로운 전통의 탄생:**

* "The claw is the law" (집게발이 법이다) 🤠
* "Yee-claw"
* "Clawntroversy" (집게발 논란)
